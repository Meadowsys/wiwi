pub struct Nom<T, M> { /* private fields */ }
Expand description

Nominal wrapper struct

This struct consists of a value T and a marker PhantomData<M>. It does not incur any overhead for the marker type; it is simply a type safe wrapper.

Newtypes are primarily created with assistance from the nom and nom_mod macros. The macros will help save you the boilerplate of writing the types and declaring unit structs to use as the marker.
