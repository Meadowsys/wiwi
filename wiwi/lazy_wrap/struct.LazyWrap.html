impl<T, F> RefUnwindSafe for LazyWrap<T, F>
where T: RefUnwindSafe, F: UnwindSafe,

RefUnwindSafe if T is RefUnwindSafe and F is UnwindSafe

Sending references of LazyWrap will send T as a reference across. F may be run and panic on the other side of the boundary if LazyWrap hasn’t been initialised yet, so must be UnwindSafe.

impl<T, F> Send for LazyWrap<T, F>
where T: Send, F: Send,

Send if T and F are both Send

Sending LazyWrap across threads will send T and F both. F may be run on the other thread if LazyWrap hasn’t been initialised/accessed yet, so F must be Send.

Sync if T is Sync and F is Send

impl<T, F> UnwindSafe for LazyWrap<T, F>
where T: UnwindSafe, F: UnwindSafe,

UnwindSafe if T and F are both UnwindSafe

Sending LazyWrap across an unwind boundary will send T and F both. T may be accessed and F may be called across an unwind boundary, and code may panic while both is happening, so both T and F must be UnwindSafe.
