fn read_bytes_ptr(&mut self, bytes: usize) -> Result<*const u8, ErrorFound>

Reads the next specified amount of bytes, returning a pointer to it if there is enough bytes left

Semantically, the returned pointer is granting shared access to the region of memory of length bytes, for lifetime 'h.

Calling this function is safe, because doing anything with the returned pointer is unsafe, and then you have to adhere to contract.

fn read_bytes_const<const BYTES: usize>( &mut self, ) -> Result<&'h [u8; BYTES], ErrorFound>

Reads BYTES bytes, returning a pointer to an array if there are enough bytes left

This is the same as read_bytes, but it returns a reference to a constant sized array, instead of a dynamically sized slice.
