Struct wiwi::serialiser_binary::SingleTypeArrayExpensive

source · 
pub struct SingleTypeArrayExpensive<T>(/* private fields */);
Available on crate feature serialiser-binary-unstable only.
Expand description

A wrapper type for items that can be serialised into a single type array.

More compute-expensive version of SingleTypeArray, for more space savings. See that struct for more info.
