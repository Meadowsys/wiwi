pub unsafe fn write_bits_u128_unchecked(&mut self, num_bits: usize, bits: u128)

Writes the specified amount of bits from the provided u128

The specified amount of bits will be read from the lower bits of the provided integer, and all other bits are silently ignored. For example, when writing 3 bits, only when the integer value of bits is less than or equal to 7 will no information be lost.

§Safety

This function will not check that num_bits is less than or equal to 128. It is undefined behaviour if the provided amount is greater than 128.
