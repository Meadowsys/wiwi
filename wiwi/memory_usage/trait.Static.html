Trait wiwi::memory_usage::Static

source · 
pub trait Static: Dynamic {
    const MEMORY_USAGE: usize;
}
Available on crate feature memory-usage-unstable only.
Expand description

Trait implemented by types that, at compile time, use a known, fixed amount of memory.

For example, u8 is always 1 byte, u64 is always 8 bytes, etc.

There is a blanket implementation of the Dynamic trait for all types that implement Static.

Note: this is not the same as Rust’s Sized trait. See the docs of the Dynamic trait for more information.

Prefer to implement this trait over Dynamic and use Dynamic over this trait where possible, for maximum flexibility.
