pub unsafe fn assume_init(self) -> ArrayChain<T, N>

Assumes all slots inside the array are initialised according to Tโs requirements, and converts into an array of T

Note: this implementation is currently subpar, as it does fully copy self into a new container using ptr::read. We have to do this because, at the time of writing:

  • transmute is a bit too dumb, and is not able to prove [T; N] and [MaybeUninit<T>; N] are guaranteed to be equal sized, even though we can see and prove it
  • transmute_unchecked is like transmute but without that compile time size check, but it is unstable, and according to a code comment will almost certainly never be stabilised (reasoning is that itโs too unsafe, too much power to give users :p, and to hopefully find other methods for doing things without it so that it isnโt needed)
  • MaybeUninit::array_assume_init is unstable (it internally makes use of transmute_unchecked)

We donโt know of any other option than to perform a ptr cast, then read from it.

ยงSafety

All slots in self must be fully initialised with valid values of T.
