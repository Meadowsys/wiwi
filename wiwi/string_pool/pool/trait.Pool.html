Expand description

Trait implemented by all pools, providing low-level manipulation of UTF-8 byte arrays in the pool. String accepts any pool that implements this trait.

Required Associated Types§

source

type Raw

Pool’s raw element. Can be anything (that is Sized), but should be a reference to, or otherwise represent one single underlying UTF-8 byte sequence.

Required Methods§

source

unsafe fn raw_from_slices(&self, slices: SlicesWrap<'_>) -> Self::Raw

Creates or retrieves a raw element for multiple slices of bytes

§Safety

The provided slices, when joined together using SlicesWrap::to_boxed_slice, must be valid UTF-8. Strictly speaking individual slices don’t have to be valid UTF-8, as long as the whole is, so this should not be relied on, even though it is quite likely that they will be.

Takes a pool’s raw element and returns the byte slice that it represents.
