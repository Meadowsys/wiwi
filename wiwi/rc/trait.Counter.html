Expand description

Trait for structs that can count references

wiwi includes two implementations: one for single threaded access (akin to std’s Rc), and the other for atomic multithreaded access (akin to std’s Arc).

§Safety

You must implement this trait correctly, as values returned from functions are directly used to control the allocation/deallocation of memory and dropping of values. In particular, returning an incorrectly low value for both strong and weak counts can lead to premature value dropping and/or deallocation, which is a memory safety issue.
